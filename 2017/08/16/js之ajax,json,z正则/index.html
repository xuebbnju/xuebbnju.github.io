所含内容是从网上的资料总结而来，并非原创，所含文字图片侵权必删！

一，ajax

1.基本概念

  • AJAX = 异步 JavaScript 和 XML。
  • AJAX 是一种用于创建快速动态网页的技术。
  • 通过在后台与服务器进行少量数据交换，AJAX 可以使网页实现异步更新。这意味着可以在不重新加载整个网页的情况下，对网页的某部分进行更新。

2.实现语法

  1. 创建 XMLHttpRequest 对象：variable=new XMLHttpRequest();

  3. POST 请求:

3.响应及事件

  • responseText属性获得字符串形式的响应数据。
  • responseXML属性获得 XML 形式的响应数据。
  • readyState 属性存有 XMLHttpRequest 的状态信息。
    • 0: 请求未初始化
    • 1: 服务器连接已建立
    • 2: 请求已接收
    • 3: 请求处理中
    • 4: 请求已完成，且响应已就绪
  • 每当 readyState 改变时，就会触发 onreadystatechange 事件。
  • status属性：200: “OK”；404: 未找到页面。

二，json

1.基本概念

  • JavaScript 对象表示法（JavaScript Object Notation）。
  • 是存储和交换文本信息的语法。

2.语法规则（ JavaScript 对象表示法语法的子集）

  • 数据在名称/值对中
  • 数据由逗号分隔
  • 花括号保存对象
  • 方括号保存数组

类别

  • JSON 值：数字（整数或浮点数），字符串（在双引号中），逻辑值（true 或 false），数组（在方括号中），对象（在花括号中），null。
  • JSON 对象：{ "firstName":"John" , "lastName":"Doe" }。

三，正则表达式

1.基本概念

  • 提供一种模式提供检索或替换的规则
  • 字面量或者reg对象

2. 三flag

  • global：全部查询完，否则查到第一个就结束
  • ignoreCase:区分大小写
  • multline:是否跨行识别

3.四个属性

global，ignorecase，multiline查询三个flag设置情况，source是正则的内容

4.对象方法

  • exec:返回匹配的内容
  • test：是否匹配
  • tostring：正则内容
  • compile：改变属性内容
  • 与字符串的方法组合使用，正则表达式当参数传进去
