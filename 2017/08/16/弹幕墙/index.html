弹幕墙实例练习知识总结

声明：

所含内容是从网上的资料总结而来，并非原创，所含文字图片侵权必删！

一，野狗云

1. 创建实例

  • 添加引用<script src="https://cdn.wilddog.com/sdk/js/2.5.8/wilddog.js"></script>。

2. 基本操作

  • ref.child('message').push(text);为子节点message添加新的键值对，text为值，key会自动创建随机的名称，按时间顺序排列。
  • ref.remove();清空所有键值和子节点。
  • ref.child("message").on("child_added",function(snapshot){})为子节点添加事件监听，增加新的键值对节点时触发。
  • snapshot.val()获取添加的键值对的值内容。

二， 实现原理

1. 基本操作原理

  • 点击发射按钮时为野狗云实例message子节点添加键值对，同时将输入框中的内容清空。
  • 按回车键时，通过trigger("click");触发发射按钮点击事件。
  • message子节点监听到添加键值对事件后，将添加的键值对的值内容取出压入保存弹幕内容的数组中。

  • 创建以此值内容为文本的弹幕div,并添加到弹幕框div中(注意要将DOM对象转化成jQuery对象):

  • 然后将此弹幕div实现运动。

    • 由于每次新添加的弹幕div只会跑一次，为了让弹幕框里一直有弹幕滚动，设置一个定时器函数，每个一段时间就从保存弹幕内容的数组中随机抽一个弹幕出来，添加弹幕div让它实现移动。
  • 点击清屏按钮时调用ref.remove();清空所有键值和子节点。并情况保存弹幕内容的数组。

2. 弹幕运用原理

  1. 弹幕div需要absolute定位，top偏移每次都比上次多50px,以防重叠，到达弹幕框底部时，再归为初始值。
  2. 左右偏移设置为right为0，让弹幕从弹幕框最右侧开始运动。
  3. 为弹幕添加动画，通过目标值设为左偏移为负的自身宽度，让弹幕div从左到右运动一次。并随机添加颜色和时间设置。
  4. 随机时间通过生产随机数实现，随机颜色通过从0-9和a-f中随机选取一个字符出来，随机地选6次组成颜色字符串。
  5. 动画的回调函数需要设置成删除此div节点，否则一直存在，浪费空间。
