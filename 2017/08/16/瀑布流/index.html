切图实例练习总结

声明：

所含内容是从网上的资料总结而来，并非原创，所含文字图片侵权必删！

一，样式设置

  1. 所有装照片的div宽度一致，高度auto。
  2. 所有装照片的div会设置为absolute,所以父元素需要relative。
  3. 计算出可以放置的列数，然后设置最外层盒子的宽度。

二，实现思路

  1. 遍历所有的照片div，对它们进行定位设置。
  2. 建立一个数组，储存第一排元素的outerHeight,作为每一列的高度初始值.
  3. 从第二排的第一个照片div开始，找出现在高度最小的那一列，便通过设置样式将元素放在这一列。
  4. 更新这一列的高度值。
  5. 储存每列高度值的数组需要在循环函数之外定义，否则会成为循环函数的内部变量，每次都会重新定义赋值。
  6. 当滑动到最后一个照片一半的时候，创建一排新的照片div，然后循环以上步骤。

三，各种高宽值区别

  1. 通过style.width(style.height)或者jQuery的$(divObj).width()/$(divObj).height()获取/设置元素的宽高
  2. 若要获取元素包含边框的宽度，则可通过divObj.offsetWidth/divObj.offsetHeight或jQuery的$(divObj).outerWidth(false)/$(divObj).outerHeight(false)获取
  3. 通过$(divObj).outerWidth(true)/$(divObj).outerHeight(true)获取带外边距的宽度

  4. 通过$(divObj).innerWidth()/$(divObj).innerHeight()获取不包含边框、不包含外边距的宽度
