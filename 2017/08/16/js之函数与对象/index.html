js之对象与函数与对象复习总结

声明：

所含内容是从网上的资料总结而来，并非原创，所含文字图片侵权必删！

一，对象

1.概念

  • 若干属性的集合。属性时一对（key，value），key必须是字符串的。
  • 方法也是一种属性。因为它的属性表示为键值对的形式。
  • javascript中的对象可以任意的扩展属性，没有class的约束。

2. 创建

  1. 字面量 ：var obj={x:1;y:2}（这种方法创建的对象的原型默认指向object.prototype）。
  2. new构造器：prototype指向对象创建器的那个函数的prototype。
  3. obj.create ：Object.create(proto [, propertiesObject ]) 是E5中提出的一种新的对象创建方式，第一个参数是要继承的原型，如果不是一个子函数，可以传一个null，第二个参数是对象的属性描述符，这个参数是可选的。var obj=object.creat({x:1;y;2}),prototype是{x:1;y:2},最终都会指向object.prototype。

注意一点：如果，var obj=object.creat(null ),prototype就不能指向object.prototype，就无法用tostring等方法。

3. 属性操作

  • 属性读写：obi.x或obj[‘x’],或者for in（可能会访问原型链上的属性,顺序不确定）。
  • 属性删除：delete之后变为undefined，有些属性不允许被删除，有权限，比如object.prototype, 全局变量，局部变量，函数变量也都不可以被删除（隐式不用var定义的可以被删除，eval中的变量也可以）。
  • 属性检测：in 往原型链上查找。
    • .hasOwnPrototypr：对象上这个属性是否存在。
    • .prototypeIsEnumerable：属性是否可以枚举而输出，原型链上的大部分属性默认为false。
  • getter和setter方法：，get,set是关键字，自己写完方法后，读写属性时默认直接调用写的方法。
    • 对象原型链上有某个属性并且有getter和setter方法，对象本身没有这个属性时，对这个属性进行赋值操作不会像平常那样在这个对象内创建这个属性，而是调用set方法对原型链上的那个属性操作。
    • 即便没有getter和setter方法，若原型链上的那个属性writable为false，那么对象本身也无法添加这个属性。
    • 如果想在这个对象内部创建那个同名属性，只能通过definePropotry方法。

4. 属性标签

  • getOwnPropotryDescriptor方法获取属性标签。
  • object.definePropotry：增加属性并设置标签，是否可以枚举enumerable，可写writable等标签，默认为false。
  • 其他方法创建的属性的标签都默认为true。
  • object.key(obj),获取obj的所有属性的key。
  • object.definePropotries：定义多个属性。

5. 对象标签

  • proto标签
  • class标签：无法直接查看，可以通过Object.prototype.toString（即便参数是值类型也会转化成对象处理）获取。
  • extensible标签：是否可以扩展属性。
    • Object.isExtensible(obj)是否可以添加。
    • Object.preventExtensions(obj)阻止对象扩展属性，而不影响现有属性的标签。
    • Object.seal(obj)让对象所有属性configurable: false。Object.isSealed(obj);。
    • Object.freeze(obj)让对象所有属性configurable: false,, writable: false。Object.isFrozen(obj);。
    • 以上操作不影响原型链，除非遍历原型链把所有原型都设置一遍。

6. 序列化

  • JSON.stringify(obj)：把js对象转化为字符串（json格式）。
  • 属性值为undefine会把属性不存在处理，NaN,Infiity变null。
  • JSON.parse('{"x" : 1}')把字符串解析成js对象。
  • 可以为属性自定义toJSON方法。
  • 可以为对象自定义toString方法。
  • valeOf尝试把对象转换为基本类型是调用，也可以自定义重写。
  • 先找valeOf，之后找toString。

二，函数

1. 概念

  • 函数是一块JavaScript代码,被定义一次,但可执行和调用多次。
  • JS中的函数也是对象,所以JS函数可以像其它对象那样操作和传递所以我们也常叫JS中的函数为函数对象。
  • 函数就是包裹在花括号中的代码块，前面使用了关键词 function。

2. 局部变量与全局变量

  • 在 JavaScript 函数内部声明的变量（使用var）是局部变量，所以只能在函数内部访问它。（该变量的作用域是局部的）。
  • 您可以在不同的函数中使用名称相同的局部变量，因为只有声明过该变量的函数才能识别出该变量。
  • 只要函数运行完毕，本地变量就会被删除。
  • 在函数外声明的变量是全局变量，网页上的所有脚本和函数都能访问它。
  • 如果您把值赋给尚未声明的变量，该变量将被自动作为全局变量声明。
  • 无var的变量一律是全局的无论在哪写的。

3. 创建

  • 函数声明：fun add(){}。
  • 函数表达式：var add=fun(){} ;。
  • 命名函数表达式:var add=fun foo(){}。
    学习文章
  • 函数构造器：var func = new Function('a', 'b', 'console.log(a + b);');。
  • (fun(){})()：直接调用，立即执行。
  • 函数前面加括号和!,是为了让声明变成表达式，不被前置，且立即执行

4. 函数属性

  • .length：参数个数。
  • .name：函数名。
  • arguments,是一个类数组，没有join那些普通数组的操作函数，我的理解是，函数传进来参数之后，会自动创建一个arguments类数组来储存这些实参（严格模式下不成立）。
  • arguments.length:实参个数 。
