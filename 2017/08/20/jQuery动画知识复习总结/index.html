jQuery动画知识复习总结

发表于 2017-08-20

摘要：总结了jQuery基本的几种动画效果的操作方法以及一些其它方面的核心操作方法。

声明：

所含内容是从网上的资料总结而来，并非原创，所含文字图片侵权必删！

一，隐藏和显示

  • .hide( options )：当提供hide方法一个参数时，.hide()就会成为一个动画方法。.hide()方法将会匹配元素的宽度，高度，以及不透明度，同时进行动画操作。
  • show()：让元素从隐藏到显示。
  • show与hide方法是修改的display属性，通过是visibility属性布局需要通过css方法单独设置。
  • 如果使用!important在你的样式中，比如display: none !important，如果你希望.show()方法正常工作，必须使用.css('display', 'block !important')重写样式。
  • .toggle( [duration ] [, complete ] )：同样的提供了时间、还有动画结束的回调。
  • toggle(display)：确定是使用show还是hide方法。

二，下拉动画

  • .slideDown()方法将给匹配元素的高度的动画，这会导致页面的下面部分滑下去，弥补了显示的方式。
  • .slideUp()：找到元素的高度，然后采用一个下滑动画让元素一直滑到隐藏，当高度为0的时候，也就是不可见的时，修改元素display 样式属性被设置为none。
  • slideToggle()：这是最基本的操作，获取元素的高度，使这个元素的高度发生改变，从而让元素里的内容往下或往上滑。

三，淡入淡出

  • fadeOut()：函数用于隐藏所有匹配的元素，并带有淡出的过渡动画效果。
  • fadeIn()函数用于显示所有匹配的元素，并带有淡入的过渡动画效果。
  • fadeToggle()：如果元素当前是可见的，则将其隐藏(淡出)；如果元素当前是隐藏的，则使其显示(淡入)。
  • .fadeTo( duration, opacity ,callback)：淡入淡出效果达到设置为给定的不透明度时停止。

四，动画

  • 语法：

  • 要特别注意所有用于动画的属性必须是数字的

  • 特别注意单位，属性值的单位像素（px）,除非另有说明。单位em 和 %需要指定使用。

  • 除了定义数值，每个属性能使用show, hide, 和 toggle。这些快捷方式允许定制隐藏和显示动画用来控制元素的显示或隐藏

  • step： fx返回的是和css相关的属性；

  • progress： fx返回的是动画进度本身的属性；
  • 动画停止：.stop( [queue ], [ clearQueue ] ,[ jumpToEnd ] )。
    • stop()：只会停止第一个动画，第二个第三个继续。
    • stop(true)：删除所有动画永远不会运行。
    • stop(true,true)： 当前动画将停止，但该元素上的 CSS 属性会被立刻修改成动画的目标值。

五，核心方法

  • jQuery.each(array/object, callback )：for循环方法的一个包装，如果返回值为false，则停止循环(相当于普通循环中的break)。
  • jQuery.inArray( value, array ,[ fromIndex ] )：传递一个检测的目标值，然后传递原始的数组，可以通过fromIndex规定查找的起始值，默认数组是0开始。
  • jQuery.trim()函数用于去除字符串两端的换行符，空格(包括连续的空格)和制表符（tab）,中间的不会去除
  • get()方法获取需要单独操作合集中的的某一个元素,获取的dom对象(注意转化成jQuery对象)。
  • index() ：参数接受一个jQuery或者dom对象作为查找的条件，搜索给定元素的索引值。
  • 如果不传递任何参数给 .index()方法，则返回值就是jQuery对象中第一个元素相对于它同辈元素的位置。
